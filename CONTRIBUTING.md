### Guide to Contributing

Thank you for your interest in this project!
We welcome your contributions, whether they be bug reports or feature suggestions.

## 1. Reporting an Issue
1. If you find a bug or would like to suggest a new feature, please create an issue by following the steps below.
2. First, check to see if an issue with the same content has already been created.
3. Click “New issue” under the “Issues” tab at the top of the screen.
Please provide detailed information about the issue, including the problem description, steps to reproduce it, and the expected behavior.
It is fine to include code to reproduce the issue, but please do not include code for a proposed fix.

## 2. Code of Conduct
To ensure a pleasant development environment for all participants, please communicate respectfully and courteously with one another.

### 開発への参加ガイド

このプロジェクトに関心を持っていただき、ありがとうございます！
バグ報告や機能提案など、皆さまからのご協力をお待ちしています。

## 1. 課題（Issue）の報告
バグを発見した場合や、新しい機能を提案したい場合は、以下の手順で Issue を作成してください。
1. 既に同じ内容の Issue が作成されていないか確認します。
2. 画面上部の「Issues」タブから「New issue」をクリックします。
3. 問題の内容、再現手順、期待される動作を詳しく記載して送信してください。
Issueには再現のためのコードは良いですが、修正案のコードは書かないでください。

## 2. 行動規範（Code of Conduct）
すべての参加者が気持ちよく開発できるよう、お互いに敬意を払った丁寧なコミュニケーションをお願いします。
